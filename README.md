# FastInBox Docs

Repositorio de documentacao tecnica publica do FastInBox.

## Site oficial

- https://fastinbox-repo.github.io/docs/

## Objetivo

Concentrar documentos formais de produto, arquitetura e operacao para apresentacao academica e continuidade de desenvolvimento.

## Estrutura principal

- `index.html`: hub de documentacao
- `styles.css`: identidade visual neo-brutal P&B
- `script.js`: interacoes de navegacao
- `documents/`: biblioteca de documentos completos
